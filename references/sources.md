# Sources — evidence base

Все механики скилла с источниками. База собрана двумя дипресёрч-прогонами
(2026-07-03/04, 435 агентов суммарно, 91 источник выкачан, каждый факт прошёл
3-голосую адверсариальную проверку). Vote = голоса «подтверждён-опровергнут».

## Отклонено адверсариальной проверкой (НЕ используется в скилле)

- «Грубость улучшает точность: Very Rude 84.8% vs Very Polite 80.8%» ([arxiv.org/abs/2510.04950](https://arxiv.org/abs/2510.04950)) — vote 0-2, противоречит корпусу остальных работ; в советах не используется.
- «Письменные навыки — независимый предиктор успеха вайбкодинга» (из 2603.14133) — vote 0-3, клейм не пережил проверку (в отличие от CS-achievement из той же работы); из советов удалён.
- «Позитивные эмоции в промпте улучшают точность, но растят сикофантию» ([arxiv.org/abs/2604.07369](https://arxiv.org/abs/2604.07369)) — vote 1-2, слабое подтверждение.
- D&D Beyond «12 классов с двухчастной формулой названий» — vote 0-3; вместе с решением юзера класс-система выпилена целиком.
- «Context rot: точность деградирует с ростом контекста» (anthropic engineering) — vote 1-2; используется только подтверждённое определение «контекст — конечный ресурс».
- «Gemini 2.0 Flash вообще нечувствительна к тону» (2512.12812) — vote 1-2, не используется.
- «Перевёрнутая U вежливости на EN/ZH/JA» (arxiv 2402.14531) — vote 1-2; используется только «оптимум зависит от языка» (3-0). Русский не исследован ни одной работой базы.

## Методология шкалы

По направлениям «нормировка поведенческих метрик» и «готовые RU/EN лексиконы» ни один
внешний факт не пережил проверку — SCALE v1 (формулы, пороги, лексиконы) спроектирована
с нуля и откалибрована на реальном корпусе 5200+ реплик; это зафиксировано в
[scale.md](scale.md) как версионируемый контракт.
